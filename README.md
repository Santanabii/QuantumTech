# QuantumTech
A modern, responsive online electronics store built with pure HTML and CSS styling.

Beautiful product cards, hover effects, filters (UI only), and a fixed shopping cart sidebar – perfect as a learning project or starting point for a frontend e-commerce template.
Features:
- Responsive product grid (works great on mobile, tablet & desktop)
- Clean & modern card design with hover lift + shadow effects
- Simple sort & category filter dropdowns (static UI – looks interactive)
- Fixed shopping cart sidebar (shows on the right / bottom on mobile)
- Attractive blue gradient header
- Easy-to-customize colors, prices, and product images
